# E-Commerce Sales Dashboard (Power BI)

## Project Overview

This project is an E-Commerce Sales Performance Dashboard built using Power BI to analyze business performance using 2022 sales data.

The dashboard provides insights into sales, profit, quantity, profit margin, customer segments, product categories, regions, and shipping types to support data-driven decision-making.

## Business Problem

Stakeholders needed a single interactive dashboard to:

* Monitor overall sales and profitability
* Identify growth or decline in performance
* Analyze top and underperforming products
* Understand regional and shipping trends

Raw data alone was not sufficient, so a visual analytics solution was required.

## Data Used

* E-commerce sales dataset (Excel)

  * Order Date, Sales, Profit, Quantity
  * Category \& Sub-Category
  * Customer Segment
  * Region \& State
  * Shipping Type

* US State latitude \& longitude file (for map visualization)

## Data Preparation

* Cleaned and validated data
* Created a calendar table for time-based analysis
* Built relationships between tables
* Created DAX measures:

  * YTD Sales
  * YTD Profit
  * YTD Quantity
  * Profit Margin
  * YoY comparisons

* Dashboard filtered to Year 2022

## Key KPIs

* YTD Sales
* YTD Profit
* YTD Quantity
* Profit Margin
* Year-over-Year (YoY) performance

## Dashboard Features

* KPI cards for performance tracking
* Sales analysis by category and product
* Top \& bottom products analysis
* Regional and state-wise sales map
* Customer segment analysis
* Shipping type performance
* Interactive filters for self-service analysis

## Business Value

This dashboard helps decision-makers:

* Monitor business performance efficiently
* Identify high-performing products and regions
* Detect underperforming areas early
* Optimize pricing, promotions, and inventory planning
